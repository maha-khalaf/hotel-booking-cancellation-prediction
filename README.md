# Hotel Booking Cancellation Prediction

## 📌 Overview
This project analyzes hotel booking data to predict whether a booking will be canceled. The goal is to help hotels reduce revenue loss by identifying high-risk cancellations.

## 🧠 Problem
Hotel cancellations lead to:
- Revenue loss
- Increased operational costs
- Last-minute price reductions

This project builds a machine learning model to predict cancellations in advance.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Key Steps
- Exploratory Data Analysis (EDA)
- Data preprocessing and feature engineering
- Decision Tree model building
- Model evaluation using:
  - Confusion Matrix
  - ROC Curve
  - AUC Score
- Hyperparameter tuning:
  - GridSearchCV
  - RandomizedSearchCV

## 📈 Results
- Built a classification model to predict booking cancellations
- Improved model performance using hyperparameter tuning
- Identified key factors influencing cancellations

## 🔍 Key Insights
- Lead time and previous cancellations strongly impact booking behavior
- Pricing and special requests influence cancellation likelihood
